One to Many

Turn a spreadsheet and the HTML page for the first row into a complete website.



No DB

Avoid database, server, and cross-domain API weirdness. Currently supports: CSV and GeoJSON. Later: KML and other formats



Whatever you want to make

Write row/index.html, and any JavaScript maps, charts, and visualizations work for the other rows.

For Example:

Data

Hello World,40,-80
Hello Row 2,45,-100
Hello Row-to-Scoped,30.429,30.4

Code

var center = replaceRow([ 40, -80 ]);
var map = L.map("map").setView( center, 6 );

var message = replaceRow("Hello World");
L.marker( center ).addTo(map);
  • /row?page=1 is the same page
  • /row?page=2 maps the second marker
  • /row?page=3 maps the third marker
